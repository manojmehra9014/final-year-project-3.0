HTML Heading

Heading 1

The h1 element is used to define the most important heading on a web page. It should be used sparingly, as it represents the main topic or purpose of the page.

Heading 2

The h2 element is used to define a sub-heading on a web page. It should be used to introduce a new section or topic that is related to the main heading.

Heading 3

The h3 element is used to define a sub-sub-heading on a web page. It should be used to introduce a new sub-section or topic that is related to the previous heading.

Heading 4

The h4 element is used to define a sub-sub-sub-heading on a web page. It should be used to introduce a new sub-sub-section or topic that is related to the previous heading.

Heading 5

The h5 element is used to define a sub-sub-sub-sub-heading on a web page. It should be used to introduce a new sub-sub-sub-section or topic that is related to the previous heading.

Heading 6

The h6 element is used to define the least important heading on a web page. It should be used to introduce a minor topic or section that is related to the previous heading.
