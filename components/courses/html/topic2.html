HTML Basics

Basics of HTML

HTML documents are structured using tags. A tag is a keyword enclosed in angle brackets that specifies how text should be displayed on a web page. The basic structure of an HTML document consists of an opening tag, the content, and a closing tag. For example, the <p> tag is used to define a paragraph of text:

<p>This is a paragraph of text.</p>

HTML documents also have a head section and a body section. The head section contains metadata about the document, such as the title and links to stylesheets, while the body section contains the content of the document.

Common HTML Tags

HTML has many tags that can be used to structure content and add meaning to web pages. Here are some of the most commonly used tags:

  • <h1> - Defines a heading
  • <p> - Defines a paragraph
  • <a> - Defines a hyperlink
  • <img> - Defines an image
  • <ul> - Defines an unordered list
  • <ol> - Defines an ordered list
  • <li> - Defines a list item
  • <div> - Defines a section of a document

HTML Attributes

HTML tags can also have attributes, which are used to provide additional information about the tag. For example, the <a> tag is used to create hyperlinks and has an href attribute that specifies the URL of the link: 

<a href="https://www.example.com">This is a link</a>

Other common attributes include src for specifying the source of an image and class for applying styles to an element with CSS.

Conclusion

HTML is an essential part of web development and provides a way to structure content and add meaning to web pages. By using tags and attributes, you can create rich, interactive web pages that engage your users and provide a great user experience.
