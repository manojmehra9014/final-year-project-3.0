File System Operations in Node.js

Node.js provides a built-in module called 'fs' that allows you to work with the file system, perform file-related operations, and manage directories...

Key file system operations include:

  • Reading Files: Using the 'fs.readFile' method to read the contents of a file.
  • Writing Files: Using the 'fs.writeFile' method to create or overwrite a file.
  • Appending to Files: Using the 'fs.appendFile' method to add content to an existing file.
  • Renaming and Deleting Files: Using 'fs.rename' and 'fs.unlink' for renaming and deleting files.
  • Working with Directories: Using 'fs.mkdir' and 'fs.readdir' for creating and listing directories.

Code Example: Reading and Writing Files

Here's an example of reading and writing files using the 'fs' module:

// Import the fs module
        const fs = require('fs');
        
        // Read a file
        fs.readFile('example.txt', 'utf8', (err, data) => {
            if (err) {
                console.error('Error reading file:', err);
                return;
            }
            console.log('File content:', data);
        });
        
        // Write to a file
        const contentToWrite = 'Hello, this is a new content.';
        fs.writeFile('newfile.txt', contentToWrite, 'utf8', (err) => {
            if (err) {
                console.error('Error writing file:', err);
                return;
            }
            console.log('File written successfully.');
        });

This code demonstrates reading the contents of a file using 'fs.readFile' and writing content to a new file using 'fs.writeFile'.
