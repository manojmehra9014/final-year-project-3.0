Working with Variables and Data Types

In JavaScript, variables are used to store and manipulate data. Understanding how to work with variables and data types is crucial for writing effective JavaScript code.

JavaScript supports several data types, including numbers, strings, booleans, arrays, objects, null, and undefined. Here's an overview of some commonly used data types:

  • Number: Represents numeric values, such as integers and decimals. Example: let age = 25;
  • String: Represents sequences of characters. Enclosed in single or double quotes. Example: let name = 'John';
  • Boolean: Represents either true or false. Example: let isStudent = true;
  • Array: Represents an ordered collection of values. Example: let numbers = [1, 2, 3, 4, 5];
  • Object: Represents a collection of key-value pairs. Example: let person = { name: 'John', age: 25 };
  • null: Represents the absence of value. Example: let value = null;
  • undefined: Represents a variable that has been declared but not assigned a value. Example: let variable;

Code Example: Working with Variables

Here's an example that demonstrates how to declare and use variables of different data types in JavaScript: 

        
<script>


    // Number
    let age = 25;

    // String
    let name = 'John';

    // Boolean
    let isStudent = true;

    // Array
    let numbers = [1, 2, 3, 4, 5];

    // Object
    let person = { name: 'John', age: 25 };

    // null
    let value = null;

    // undefined
    let variable;

    console.log(age);
    console.log(name);
    console.log(isStudent);
    console.log(numbers);
    console.log(person);
    console.log(value);
    console.log(variable);

    
</script>

In the above code, we declare variables of different data types and assign them values. Then, we use the console.log() function to display the values of these variables in the browser's console.

By understanding variables and data types, you'll be able to store and manipulate data effectively in your JavaScript programs.
