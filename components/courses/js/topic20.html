Making HTTP Requests with Fetch API

The Fetch API provides an easy and modern way to make HTTP requests from JavaScript. It offers a powerful and flexible approach to handle network requests and interact with APIs.

With the Fetch API, you can send HTTP requests such as GET, POST, PUT, DELETE, and more, and handle the response asynchronously using Promises.

Code Example: Fetching Data from an API

Here's an example that demonstrates how to use the Fetch API to fetch data from an API endpoint: 

                
                    const apiUrl = 'https://api.example.com/data';
                    
                    fetch(apiUrl)
                        .then(response => response.json())
                        .then(data => {
                            console.log(data);
                            // Process the received data
                        })
                        .catch(error => {
                            console.log('An error occurred:', error);
                            // Handle the error
                        });

In the above code, we use the `fetch()` function to send an HTTP GET request to the specified API URL. The response is returned as a Promise, which we can then use to parse the response using the `json()` method. Finally, we handle the data in the subsequent `then()` block or catch any errors that occurred during the request.

The Fetch API also allows you to customize the request headers, handle authentication, send data in the request body, and more. It provides a flexible and modern approach to working with HTTP requests in JavaScript applications.
