Arrays and Array Manipulation

Arrays are fundamental data structures in JavaScript that allow you to store and manipulate collections of values. Understanding how to work with arrays and perform array manipulation is crucial for building dynamic and data-driven applications.

In JavaScript, an array is a special type of object that holds an ordered list of values, which can be of any data type. Arrays provide various methods and properties that enable you to add, remove, access, and modify elements within the array.

Code Example: Manipulating an Array

Here's an example that demonstrates how to manipulate an array in JavaScript: 

                
<script>
    let fruits = ["apple", "banana", "orange"];
        
    // Adding an element to the array
    fruits.push("grape");
        
    // Removing an element from the array
    let removedFruit = fruits.pop();
        
    // Accessing an element in the array
    let secondFruit = fruits[1];
        
    console.log(fruits); // Output: ["apple", "banana", "orange"]
    console.log(removedFruit); // Output: "grape"
    console.log(secondFruit); // Output: "banana"
</script>

In the above code, we create an array called fruits and add three elements using the push method. We then remove the last element using the pop method, store it in the removedFruit variable. We also access the second element in the array using index notation and store it in the secondFruit variable. The resulting array and values are printed to the browser's console.

By mastering array manipulation techniques, you can efficiently work with collections of data, perform operations like sorting and filtering, and build more dynamic and interactive JavaScript applications.
