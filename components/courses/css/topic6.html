CSS Color and Background

Color

CSS provides a variety of ways to set the color of text and other elements. Here are some examples:

  • Red text
  • Green text
  • Blue text

To set the color of an element using CSS, you can use the color property. Here's an example:

color: #f00;

Background

You can also set the background color and image of an element using CSS. Here's an example:

background-color: #f5f5f5;
background-image: url("background.jpg");
background-size: cover;
