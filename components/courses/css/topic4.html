CSS Display and Positioning

Display Property

The display property in CSS controls how an element is rendered on a web page. There are several values for this property:

  • block: displays the element as a block-level element, taking up the entire width of its container
  • inline: displays the element as an inline-level element, taking up only as much width as necessary to display its content
  • inline-block: displays the element as an inline-level block container
  • none: hides the element completely

Positioning

The position property in CSS controls how an element is positioned on a web page. There are several values for this property:

  • static: the default position, which means the element is positioned according to the normal flow of the document
  • relative: the element is positioned relative to its normal position, using the top, right, bottom, and left properties
  • absolute: the element is positioned relative to its nearest positioned ancestor, using the same properties as relative
  • fixed: the element is positioned relative to the viewport, using the same properties as absolute
