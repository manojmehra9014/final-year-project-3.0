CSS Grid

What is CSS Grid?

CSS Grid is a layout system that allows you to create complex grid-based layouts with ease.

How does CSS Grid work?

CSS Grid works by dividing a web page into a grid of rows and columns, which can then be used to place and position content.

CSS Grid Properties

  • display: grid; - Defines the container as a grid container.
  • grid-template-columns - Defines the number and size of columns in the grid.
  • grid-template-rows - Defines the number and size of rows in the grid.
  • gap - Defines the spacing between grid items.
  • justify-content - Defines how grid items are horizontally aligned within the grid.
  • align-items - Defines how grid items are vertically aligned within the grid.
  • grid-row - Defines which row(s) an item should occupy.
  • grid-column - Defines which column(s) an item should occupy.
