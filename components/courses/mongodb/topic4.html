Querying Data with MongoDB

MongoDB provides a flexible and powerful query language that allows you to retrieve data from your collections. Here are some techniques for querying data in MongoDB:

Filtering Documents

You can use the find() method to filter documents based on specific criteria. For example:


  // Find all documents with age greater than 30
  db.collection('myCollection').find({ age: { $gt: 30 } });

Sorting Results

To sort the results of your query, you can use the sort() method. Here's an example:


  // Sort documents by age in descending order
  db.collection('myCollection').find().sort({ age: -1 });

Limiting Results

If you want to limit the number of documents returned by a query, you can use the limit() method. For example:


  // Limit the result to only 10 documents
  db.collection('myCollection').find().limit(10);

These are just a few examples of how you can query data in MongoDB. The query language offers many more features and operators to help you retrieve the information you need.
