CRUD Operations in MongoDB

MongoDB is a document-oriented NoSQL database that supports CRUD operations (Create, Read, Update, Delete) on data. Let's explore each of these operations:

Create (Insert) Documents

To create new documents in MongoDB, you can use the insertOne() or insertMany() methods. Here's an example of inserting a document:


  db.collection('myCollection').insertOne({
    name: 'John Doe',
    age: 30,
    email: 'johndoe@example.com'
  });

Read (Query) Documents

To query and retrieve data from MongoDB, you can use the find() method. Here's an example of querying documents:


  db.collection('myCollection').find({ age: { $gt: 25 } });

Update Documents

To update existing documents in MongoDB, you can use the updateOne() or updateMany() methods. Here's an example of updating a document:


    db.collection('myCollection').updateOne(
    { name: 'John Doe' },
    { $set: { age: 35 } }
  );

Delete Documents

To delete documents from MongoDB, you can use the deleteOne() or deleteMany() methods. Here's an example of deleting a document:


  db.collection('myCollection').deleteOne({ name: 'John Doe' });

These are the basic CRUD operations in MongoDB. By using these operations, you can manipulate data in your MongoDB collections effectively.
