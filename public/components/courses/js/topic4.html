Operators and Expressions in JavaScript

Operators and expressions are fundamental concepts in JavaScript that allow you to perform operations and manipulate data. Understanding different types of operators and how to use them is essential for writing effective JavaScript code.

JavaScript provides various types of operators, including arithmetic, assignment, comparison, logical, and more. Here's an overview of some commonly used operators:

  • Arithmetic Operators: Used to perform mathematical calculations, such as addition (+), subtraction (-), multiplication (*), division (/), and modulus (%).
  • Assignment Operators: Used to assign values to variables, such as assignment (=), addition assignment (+=), subtraction assignment (-=), and more.
  • Comparison Operators: Used to compare values, such as equal to (==), not equal to (!=), greater than (>), less than (<), and more.
  • Logical Operators: Used to combine and manipulate logical values, such as logical AND (&&), logical OR (||), and logical NOT (!).
  • Conditional (Ternary) Operator: Used for conditional expressions. It has the syntax condition ? expression1 : expression2.

Code Example: Using Arithmetic Operators

Here's an example that demonstrates how to use arithmetic operators in JavaScript: 

        
            <script>
                let x = 5;
                let y = 3;

                let addition = x + y;
                let subtraction = x - y;
                let multiplication = x * y;
                let division = x / y;
                let modulus = x % y;

                console.log(addition);       // Output: 8
                console.log(subtraction);    // Output: 2
                console.log(multiplication); // Output: 15
                console.log(division);       // Output: 1.6666666666666667
                console.log(modulus);        // Output: 2
            </script>

In the above code, we declare two variables (x and y) and perform arithmetic operations using the arithmetic operators. The results are then printed to the browser's console using the console.log() function.

By understanding operators and expressions, you'll be able to perform calculations and manipulate data effectively in your JavaScript programs.
