CSS Animations and Transitions

What are CSS Animations?

CSS animations are a way to create animated effects on web pages using CSS. Animations can be used to make elements move, change size or shape, fade in or out, and more.

What are CSS Transitions?

CSS transitions are a way to create smooth transitions between different states of an element. Transitions can be used to change the color, size, or position of an element over time.

Example of a CSS Transition

Here is an example of a CSS transition that changes the background color of a box when the user hovers over it:

touch me!
        
.box {
    width: 100px;
    height: 100px;
    background-color: blue;
    transition: background-color 1s ease-out;
}
.box:hover {
    background-color: red;
}

Example of a CSS Animation

Here is an example of a CSS animation that spins a circular element:


@keyframes spin {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}
.spinner {
  width: 100px;
  height: 100px;
  border: 10px solid black;
  border-top: 10px solid blue;
  border-radius: 50%;
  animation: spin 1s linear infinite;
}

These examples are just a small taste of what is possible with CSS animations and transitions. By combining CSS with JavaScript, you can create even more complex and dynamic animations on your web pages.
