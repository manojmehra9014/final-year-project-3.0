CSS Flexbox

Flexbox is a layout module in CSS3 that provides a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown or dynamic. Here are some of the key features of CSS flexbox:

  • Flex containers and flex items
  • Direction of the main axis
  • Alignment along the main axis and cross axis
  • Flexibility of flex items
  • Wrapping flex items to multiple lines
  • Ordering of flex items
  • Responsive design with media queries

Basic Flexbox Concepts

Flexbox works by dividing the available space within a container into flexible boxes, which can then be manipulated using various properties. The main concepts to understand are:

  • Flex container: the parent element that contains the flex items.
  • Flex item: the child elements that are contained within the flex container.
  • Flex axis: the main axis along which the flex items are arranged (either horizontally or vertically).
  • Flex direction: the direction in which the flex items are arranged (either row or column).
  • Flex wrap: whether the flex items should wrap to a new line if they exceed the width of the container.
  • Flex flow: shorthand property for specifying the flex direction and flex wrap properties.
  • Justify content: how the flex items are aligned along the main axis.
  • Align items: how the flex items are aligned along the cross axis.
  • Align content: how multiple lines of flex items are aligned along the cross axis.
